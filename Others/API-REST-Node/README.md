# A simple API REST using Node and Express

## **Run**

Install dependencies:

```yarn install```

Run API:

```yarn serve```

To testing endpoits with [Postman](https://www.postman.com/), import `APIClients.postman_collection.json`.

## **Endpoints**

- ***GET*** all: `/clients`
- ***GET*** a specifc client: `/clients/{id}`
- ***POST***: `/clients`
- ***PUT***: `/clients/{id}`
- ***DELETE***: `/clients/{id}`
